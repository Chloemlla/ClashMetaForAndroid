# ClashMetaForAndroid — Trellis workflow (subset)

This repo uses a **Trellis-lite** layout for product specs that must stay in-tree:

- `spec/android/` — Android product/UX contracts (e.g. update-notes gate)
- `scripts/` — generators used by agents/CI (no local Gradle required for notes)

Full Janus Trellis install is optional via `Install-CodexTrellis.ps1`.
